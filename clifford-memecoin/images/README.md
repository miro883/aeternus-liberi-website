# Images Directory

This directory contains all the image assets for the Clifford memecoin website.

## Required Images

You need to add the following images to make the website fully functional:

### Main Logo & Branding
- `clifford-logo.png` (40x40px) - Main logo for navigation
- `clifford-hero.png` (250x250px or larger) - Hero section image of Clifford

### Wallet & Currency Icons
- `bnb-icon.png` (32x32px) - BNB currency icon
- `metamask-icon.png` (32x32px) - MetaMask wallet icon
- `trust-wallet-icon.png` (32x32px) - Trust Wallet icon

## Image Guidelines

### Clifford Photos
- Use high-quality photos of Clifford from the Instagram account
- Ensure you have proper rights to use the images
- Recommend 1:1 aspect ratio (square) for hero images
- Use JPG or PNG format, optimized for web

### Logo Design
- Create a circular logo with Clifford's image
- Add "$CLIFFORD" text treatment if needed
- Use PNG with transparent background
- Maintain consistent branding colors

### Color Scheme
The website uses these colors for consistency:
- Primary: #ff6b35 (Orange)
- Secondary: #f7931e (Golden Orange)
- Accent: #ffc107 (Yellow)
- Dark backgrounds: #0a0a0a, #1a1a1a, #2a2a2a

## How to Add Images

1. Download/create the required images
2. Optimize them for web (compress without losing quality)
3. Place them in this `/images/` directory
4. Ensure filenames match exactly as referenced in the HTML/CSS

## Image Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress images
- Keep file sizes under 500KB each for fast loading
- Use appropriate formats: PNG for logos, JPG for photos
- Consider WebP format for better compression (with fallbacks)
